 # Estado de Situación y Mapa de Ruta: Proyecto de Soberanía Integral

Este es el Estado de Situación y Mapa de Ruta para el proyecto de Soberanía Integral. Como Analista de Sistemas Lógicos, este documento actúa como tu README.md maestro, consolidando la visión estratégica, la logística familiar y la arquitectura técnica para mitigar la entropía de información.

## 1. El Núcleo del Sistema (Kernel de Subsistencia)

El objetivo es garantizar la resiliencia biológica y económica del núcleo familiar para liberar carga cognitiva y permitir el desarrollo de proyectos de alto nivel.

### Estrategia Alimentaria:
Implementación de "Batch Cooking" basado en Sampa (alimento de supervivencia) y Kéfir (probiótico).

### Operación Despensa:
Optimización de compras mayoristas y ferias para maximizar el rendimiento de los ingresos actuales.

### Prioridades de Salud/Bienestar:
- **Financiamiento del tratamiento dental de Paola** (Meta: $50.000; Estado: En proceso).
- **Sustento de la educación de Emilia** ($4.500 mensuales).
- **Gestión de afiliación a ANDA** para eficiencia de costos médicos.

## 2. Red de Nodos Operativos (Infraestructura FÍsica)

El proyecto se despliega en el territorio a través de células de producción y distribución:

### Nodo Troya (Cerrito de la Victoria):
Centro de mando y logística inicial. Punto de venta de productos seleccionados (Arroz, Jugos de Transición).

### Nodo Norteamérica (El Cerro):
Operado en colaboración con Claudia. Foco en la producción de excedentes de Sampa y alimentos Detox para la venta local.

### Proyecto Habitacional "Soberanía y Convivencia":
Plan a mediano plazo para adquirir propiedad en el Oeste de Montevideo o Canelones (Presupuesto: U$S 30.000). Requisito: Espacio para huerta automatizada y gallinero moderno.

## 3. Ejes Estratégicos de Desarrollo

División de la atención del "Orquestador" para evitar la dispersión:

### Ojo de Paz (Nivel Internacional):
Arquitectura de seguridad criptográfica de bajo costo e impacto humanitario.

### Soberanía Rural (Nivel Local):
Aplicación de tecnología (Arduino/Automatización) a la producción de pequeña escala.

### Transición Laboral:
Evolución táctica de "Guardia de Seguridad" (presencia física activa) a "Sereno" (vigilancia pasiva que permita el uso de PC/Programación en tiempo real).

## 4. Arquitectura Técnica: La App de Identidad (Life OS)

Para gestionar la complejidad sin "entreverarse", se ha definido el siguiente stack tecnológico orientado a la soberanía y la rapidez de ejecución:

### Stack Tecnológico

| Componente | Tecnología | Razón |
|------------|------------|--------|
| Backend | Java + Spring Boot | Robustez lógica y maestría del usuario |
| Frontend | HTMX + Tailwind CSS | Interfaz ágil, "Antigravity" en el desarrollo, optimizada para móvil sin la carga de frameworks JS pesados |
| Base de Datos | Supabase (PostgreSQL) | Persistencia robusta y segura |

### Esquema de Datos Inicial (Tablas Maestras)

| Módulo | Entidad | Función Crítica |
|---------|----------|-----------------|
| Resiliencia | stock_soberano | Cálculo de días de autonomía alimentaria |
| Finanzas | flujo_nodos | Seguimiento de ventas y ahorro para salud (Paola) |
| Bitácora | captura_logica | Registro de ideas y código durante turnos de guardia |
